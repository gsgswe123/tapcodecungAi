Basic Data Types

A variable in C++ must be a specified data type. The data type determines what type of value the variable can store.

Data Type Size Description
int 4 bytes Stores whole numbers
float 4 bytes Stores fractional numbers (6-7 decimal digits)
double 8 bytes Stores fractional numbers (15 decimal digits)
char 1 byte Stores a single character
bool 1 byte Stores true or false values
string Varies Stores a sequence of characters

Declaring Variables

Example: Different Data Types
#include <iostream>
#include <string>
using namespace std;

int main() {
  int myNum = 5;
  float myFloatNum = 5.99f;
  double myDoubleNum = 9.98;
  char myLetter = 'D';
  bool myBoolean = true;
  string myText = "Hello";

  cout << myNum << endl;
  cout << myFloatNum << endl;
  cout << myDoubleNum << endl;
  cout << myLetter << endl;
  cout << myBoolean << endl;
  cout << myText;

  return 0;
}

Type Safety

C++ is strongly typed, meaning you cannot assign a value of one type to a variable of another type without explicit conversion.

Type Safety Example
int myInt = 9;
double myDouble = myInt;

double anotherDouble = 9.99;
int anotherInt = (int)anotherDouble;
