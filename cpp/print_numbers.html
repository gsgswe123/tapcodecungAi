Numeric Types

C++ has three main numeric types for storing numbers:

Type Size Range Precision
int 4 bytes -2,147,483,648 to 2,147,483,647 Whole numbers
float 4 bytes ±3.4e±38 6-7 decimal digits
double 8 bytes ±1.7e±308 15 decimal digits

Integer Types

Integer Example
int myNum = 1000;
cout << myNum;

Integer Variations:

Signed and Unsigned
int signedNum = -50;
unsigned int unsignedNum = 50;

short shortNum = 32000;
long longNum = 2000000000L;
long long veryLongNum = 9000000000000000000LL;

Floating Point Types

Float vs Double
float myFloat = 5.75f;
double myDouble = 19.99;

cout << myFloat << endl;
cout << myDouble;

Note: Use f suffix for float literals, otherwise they default to double.

Scientific Numbers

Floating point numbers can be expressed using scientific notation:

Scientific Notation
float f1 = 35e3f;
double d1 = 12E4;
double d2 = 87e-5;

cout << f1 << endl;
cout << d1 << endl;
cout << d2;

Practice Exercises

Exercise 1: Basic Math

Create a program that calculates the area and perimeter of a rectangle:

  • Declare variables for length (5.5) and width (3.2)
  • Calculate area (length × width)
  • Calculate perimeter (2 × (length + width))
  • Print both results

Exercise 2: Temperature Converter

Write a program to convert temperatures:

  • Declare a variable for Celsius temperature (25.0)
  • Convert to Fahrenheit: F = (C × 9/5) + 32
  • Convert to Kelvin: K = C + 273.15
  • Print all three temperatures

Exercise 3: Precision Comparison

Demonstrate the difference between float and double precision:

  • Store 3.141592653589793 in both float and double
  • Print both values and observe the difference
  • Calculate the result of dividing 10 by 3 using both types

Exercise 4: Integer Overflow

Explore integer limits:

  • Declare a short variable with value 32767
  • Add 1 to it and print the result
  • Explain what happens (integer overflow)
