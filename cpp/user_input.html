C++ User Input

The cin object is used to get input from the user. It uses the extraction operator >>

Example: Simple Input
#include <iostream>
using namespace std;

int main() {
  int x;
  cout << "Type a number: ";
  cin >> x;
  cout << "Your number is: " << x;
  return 0;
}

Creating a Calculator

Let's create a simple calculator that adds two numbers entered by the user:

Example: Calculator
#include <iostream>
using namespace std;

int main() {
  int x, y, sum;

  cout << "Enter first number: ";
  cin >> x;

  cout << "Enter second number: ";
  cin >> y;

  sum = x + y;
  cout << "Sum is: " << sum;

  return 0;
}

String Input

You can also use cin to get string input from the user:

Example: String Input
#include <iostream>
#include <string>
using namespace std;

int main() {
  string firstName;

  cout << "Enter your name: ";
  cin >> firstName;

  cout << "Hello " << firstName;

  return 0;
}

Note: cin only reads the first word. For full lines, use getline()

Full Line Input

To read an entire line including spaces, use getline():

Example: getline()
#include <iostream>
#include <string>
using namespace std;

int main() {
  string fullName;

  cout << "Enter your full name: ";
  getline(cin, fullName);

  cout << "Your name is: " << fullName;

  return 0;
}
