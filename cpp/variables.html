Declaring Variables

Variables are containers for storing data values. In C++, you must specify the type of data the variable will store.

Syntax
type variableName = value;

Common Data Types:

  • int - Stores integers (whole numbers)
  • double - Stores floating point numbers
  • char - Stores single characters
  • string - Stores text
  • bool - Stores true or false

Examples

Creating Variables
#include <iostream>
using namespace std;

int main() {
  int myNum = 15;
  double myFloat = 5.99;
  char myLetter = 'D';
  string myText = "Hello";
  bool myBoolean = true;

  cout << myNum << endl;
  cout << myFloat << endl;
  cout << myLetter << endl;
  cout << myText << endl;
  cout << myBoolean << endl;

  return 0;
}

Assigning Values

You can declare a variable without assigning a value, then assign it later:

Assign Later
int myNum;
myNum = 15;
cout << myNum;

You can also change the value of a variable:

Change Value
int myNum = 15;
myNum = 10;
cout << myNum;
